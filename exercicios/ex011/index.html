Audios

Modo fácil usando a tag <audio> e "controls" para acionar o controle de audio.


Modo difícil usando a tag <audio> seguido de source, com o type de audio.

Videos

Modo fácil usando a tag <video> podemos adicioinar diretamente usando o visual studio code.

Modo difícil usamos a tag <video> seguido do source, com o type de video, no exemplo adicionei webm,mkv,ogv e mp4 nessa mesma ordem.

OBS: Adicionei junto a tag de abertura funções como controls para adicionar os controles referentes aos arquivos(play, pause, linha do tempo) width para tamanho de video e poster para adicionar a capa do video e o loop.
